# confianza_gov
Análisis de la evolución de la confianza en el gobierno (OCDE, 2006–2024) en diálogo con la tradición clásica. Un proyecto que combina datos contemporáneos, reflexión histórica y lectura de la conjuración de Catilina (Salustio y Cicerón) para pensar la legitimidad institucional ayer y hoy.
