## Confianza institucional y evolución comparada (2006–2024)

Repositorio dedicado al análisis de la evolución de la confianza en el gobierno en distintos países a partir de datos oficiales de la OCDE.

La versión web interactiva del análisis puede visualizarse en:
<https://ecosdenuestrosclasicos.github.io/confianza_gov/>

---

### Sobre el proyecto

Este proyecto nace de una pregunta sencilla pero relevante:

**¿Cómo ha cambiado la confianza institucional en los últimos años y qué implicaciones tiene esa evolución?**

Para responderla, utilizo programación en R y datos oficiales de la OCDE (Organización para la Cooperación y el Desarrollo Económicos), trabajando el proceso completo:

- Selección y descarga del dataset
- Limpieza y transformación de datos
- Cálculo de variaciones en puntos porcentuales
- Construcción de tablas y visualizaciones
- Publicación en formato HTML reproducible

El objetivo no es solo mostrar resultados, sino hacer transparente el método.

---

### Metodología

El análisis se centra en:

- Confianza declarada en el gobierno
- Periodo completo disponible (2006–2024)
- Comparación entre España, varios países latinoamericanos (México, Chile y Colombia) y países europeos de referencia (Francia, Italia y Alemania)

Se calcula la variación total en puntos porcentuales entre el primer y el último año disponible para cada país, lo que permite observar tendencias estructurales y no solo fotografías puntuales.
